## 📘 内容说明

### 1. `doc/`

存放学习时收集的文档资料，包括语法说明、笔记、图解、读书摘要等。

### 2. `go_base/`

该目录包含大量 **Go 基础语法的 demo 代码**，内容覆盖：

* Go 基本类型与变量
* 函数、方法和值/指针接收者
* 结构体与接口
* 切片与 map
* 错误处理
* Goroutine 与 Channel
* 并发控制（sync 包）

非常适合初学者快速回顾或查询示例。

### 3. `pyp/golang-IM-System/`

一个简单的 **即时通讯（IM）系统 Demo**，用于展示 Go 在网络编程与并发场景下的应用。
该项目是我从基础语法过渡到实际应用的第一个小型练手项目。
